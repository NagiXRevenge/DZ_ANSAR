Youtube, Watsapp, Ozon, Telegram, Chrome.

Youtube: Используеся для просмотра контента: о развлекательного до познавательного. Он, правда, запрещен в РФ, но тем не менее используется миллионами.

Watsapp: Используется для общения и передачи информации друзьям, близким, коллегам. Я лично использую его для общения с друзьями и близкими, звонков без роуминга и лишних списаний.

Ozon: Используется для покупок различных товаров и является примером популярного маркетплейса. Я использую его для покупок нужных вещей: для школы, дома и для повседневных задач.
